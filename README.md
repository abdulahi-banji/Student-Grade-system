# Student Grade Management System

A Python **CLI application**(Command line interface) to manage students and their grades.  
The program allows adding students, recording grades, viewing statistics, and generating reports, all stored in a JSON file.

---

## Features

- Add new students
- View all students
- Search for a student
- Add grades to students
- Update student information
- Delete students
- Generate full report
- View statistics (average, min, max)
- Save and load data from `grades.json`
